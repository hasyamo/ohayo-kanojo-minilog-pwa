# 積み上げ前

判断を、軽く残すためのPWA。

---

## これは何か

**積み上げ前**は、  
その日の「行動」ではなく **判断** を  
Markdownとして一瞬で書き出すための、個人用PWAです。

- 保存しない
- 蓄積しない
- 管理しない

ただ **型に当てて、Markdownを生成するだけ** の道具です。

---

## なぜ作ったか（背景）

日々の開発や運用の中で、

- ログを書くこと自体が重くなる
- 正確に書こうとして手が止まる
- 記録が「成果報告」になってしまう

という状態に陥りました。

一方で、本当に後から振り返りたいのは、

- なぜその日は触らなかったのか
- なぜ様子見にしたのか
- なぜ迷っていたのか

といった **判断の理由** でした。

そこで、

- 記録は Markdown ファイル 1つに集約
- 書式は毎回考えない
- 数値や成果は扱わない

という前提で、  
**「判断だけを30秒で残す型押し器」** として  
このPWAを作りました。

---

## できること

- 日付（曜日付き）を自動生成
- モードを選択（観測 / 改修 / 仕込み / 停止 / 迷い）
- 判断を1行で入力
- 理由・タグを任意で追加
- Markdownテキストを生成してコピー

生成されたMarkdownは、  
iA Writer 等のエディタで **1つのファイルに追記** して使います。

---

## できないこと（やらないこと）

このPWAは、意図的に以下を **実装していません**。

- ログの保存
- 履歴一覧
- 編集・検索
- 数値の記録
- タスク管理

「記録する場所」はPWAではなく、  
**Markdownファイル側を正本** とする設計です。

---

## 想定する使い方

1. 思考が固まる前にPWAを開く
2. モードを選び、判断を1行書く
3. Markdownをコピー
4. `mini-log.md` に貼り付ける
5. 終わり

週末や後日、  
このログを元に振り返りや整理を行います。

---

## 出力フォーマット例

```md
2026-01-31（土）

モード：仕込み  
判断：判断の入力テスト  
理由：理由の入力テスト  
タグ：#改修 #設計 #手応え #運用
```

---

## 名前について

「積み上げノート」が  
数値や事実を積み上げるためのツールだとすれば、

**積み上げ前**は  
積み上げる前に迷っていた判断を残すための道具です。

---

## ライセンス / 利用

- 個人利用前提
- 自由に改変・再利用可
- ただし「軽さ」を失わないことを推奨
